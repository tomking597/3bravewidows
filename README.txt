3bravewidows - Static Website
=============================
This website is a tribute to our father's writings from old Malaya.

Folder Structure:
- index.html (homepage)
- writings.html (stories)
- gallery.html (images)
- about.html (project info)
- style.css (site design)
- images/ (placeholder photos)

How to View Locally:
1. Unzip this folder.
2. Open index.html in your browser.
OR
3. Run a local server:
   python -m http.server 8000
   Visit http://localhost:8000

Customizing:
- Replace images in images/ with your own.
- Edit .html files with your text editor to update content.
- Modify style.css for colors, fonts, or layout changes.
